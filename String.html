Index

字串是數組從0開始
s = "Hello, world"
print(s[0]) [] 裡面放的是index, 第一個是0

len() 回傳字串長度
s = "Hello, world"
print(len(s))


in

in() 查看字串是否在文本裡
s = "abcdefg"
print("a" in s)


循環遍歷字串

s = "Hello, world"
print(s[2:5]) 把index 2~4的字印出來，從二開始取到5(不包含5)
結果是llo，字串的第一個字index 是 0
print(s[:5])從index 0 取到 5(不包含)
print(s[2:])從index 2 取到最後一個
print(s[-5:-2]) 從倒數第5 個字取到倒數第 2個字(不包含)


切片字串

s.strip() 刪除開頭跟結尾的所有空格
s.split(", ") 將字串s 拆分為子字串 用list 包起來


修改字串

s.upper() 回傳大寫字串
s.lower() 會傳小寫字串
s.replace("A", "B") 把s 字串內的所有A 替換成B


String formate

string formate:
n = 10
s = f"n is {n}" 用{}包起來可以把數字裝在字串裡
s = f"n is {n:.2f}" 取小數到第二位
s = f"10 + 1 is {n + 1}" 也可以在{} 做運算

在""沒辦法放 " 用\"這樣就可以讓字串內有 "
print("abcdefg \" hijklmnop")
