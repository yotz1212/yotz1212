< 比較左邊的值是否小於右邊的值

if、elseif、else

    print("a")

    print("b")
else:
    print("c")

if 為True 就執行if 內的程式
else if 是if 為False 時執行，else if 可以不只有一個
else 是所有if、else if 都為False時執行
要用else if 跟else 一定要先有一個if

for loop

for i in range(1, 10):
    print(i)

i 可以自己取名字
range(start, stop, step) 內放的是數字
第一個是起始值，第二個是結束值(不包含自己)，第三個是遞增或遞減的值(可選)
for i in range(1, 10): 是印０～９

for i in range(1, 10):
    print(i)
    if i == 5:
        break
else:
    print("ABC")
如果for 沒有沒中斷的話，會執行else
